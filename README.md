## Articles

Core Articles (use all the time)
* el → le
* la → la
* los → les (masculin pluriel)
* las → les (féminin pluriel)
* un → un
* una → une
* unos → des (masculin)
* unas → des (féminin)
Quick rule
* el / la / los / las → specific (the)
* un / una / unos / unas → general (a / some)

## Pronouns

Subject Pronouns
* yo → je
* tú → tu
* él → il
* ella → elle
* nosotros / nosotras → nous
* vosotros / vosotras → vous (pluriel familier)
* ellos / ellas → ils / elles
* usted → vous (singulier formel)
* ustedes → vous (pluriel formel)
Examples (1–80)
* Yo estudio → J’étudie
* Tú hablas → Tu parles
* Él trabaja → Il travaille
* Ella vive aquí → Elle vit ici
* Nosotros comemos → Nous mangeons
* Vosotros vais → Vous allez
* Ellos leen → Ils lisent
* Usted es profesor → Vous êtes professeur
* Ustedes vienen → Vous venez
Object Pronouns (81–160)
* me → me / moi
* te → te / toi
* lo / la → le / la
* nos → nous
* os → vous
* los / las → les
Examples:
* Él me llama → Il m’appelle
* Yo te veo → Je te vois
* Ella lo conoce → Elle le connaît
* Nosotros nos ayudamos → Nous nous aidons
* Vosotros os laváis → Vous vous lavez
* Ellos los buscan → Ils les cherchent
Reflexive Pronouns (161–240)
* me → me
* te → te
* se → se
* nos → nous
* os → vous
* se → se
Examples:
* Yo me levanto → Je me lève
* Tú te duchas → Tu te douches
* Él se baña → Il se baigne
* Nosotros nos divertimos → Nous nous amusons
* Ellos se llaman → Ils s’appellent
Quick Rule
* Subject pronouns → who does the action
* Object pronouns → who receives the action
* Reflexive pronouns → action on self

## Conjunctions

* y → et
* o → ou
* pero → mais
* porque → parce que
* si → si (if)
* cuando → quand
* entonces → alors
* así que → donc
y / o (1–80)
* quiero café y pan → je veux café et pain
* quiero café o té → je veux café ou thé
* voy y trabajo → je vais et je travaille
* voy o descanso → je vais ou je me repose
* puedo y quiero → je peux et je veux
* puedo o paro → je peux ou j’arrête
* hago y aprendo → je fais et j’apprends
* hago o intento → je fais ou j’essaie
pero (81–140)
* quiero pero trabajo → je veux mais je travaille
* puedo pero fallo → je peux mais j’échoue
* voy pero tarde → j’y vais mais tard
* hago pero dudo → je fais mais je doute
* digo pero cambio → je dis mais je change
* veo pero ignoro → je vois mais j’ignore
porque (141–200)
* voy porque quiero → je vais parce que je veux
* estudio porque puedo → j’étudie parce que je peux
* trabajo porque necesito → je travaille parce que j’ai besoin
* hago porque importa → je fais parce que c’est important
* digo porque sé → je dis parce que je sais
* vuelvo porque debo → je reviens parce que je dois
entonces / así que (201–260)
* llueve entonces paro → il pleut alors j’arrête
* puedo así que voy → je peux donc j’y vais
* termino entonces descanso → je finis alors je me repose
* funciona así que sigo → ça marche donc je continue
* fallo entonces intento → j’échoue alors j’essaie
* entiendo así que cambio → je comprends donc je change
si (261–320)
* si puedo, voy → si je peux, je vais
* si quiero, hago → si je veux, je fais
* si veo, digo → si je vois, je dis
* si tengo, doy → si j’ai, je donne
* si gano, pago → si je gagne, je paie
* si pierdo, aprendo → si je perds, j’apprends
cuando (321–380)
* cuando voy, veo → quand je vais, je vois
* cuando quiero, hago → quand je veux, je fais
* cuando puedo, intento → quand je peux, j’essaie
* cuando estudio, mejoro → quand j’étudie, je m’améliore
* cuando trabajo, gano → quand je travaille, je gagne
* cuando paro, pienso → quand j’arrête, je réfléchis
